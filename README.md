# Custom Multi-Page Website (From Scratch)

This project is a fully hand-coded website developed from scratch for professional purposes, without using any CMS or page builders.  
It demonstrates core front-end development skills, JavaScript logic, and structured multi-page design.

## Project Overview
The website was designed and implemented entirely by hand using HTML, CSS, and JavaScript.  
It features multiple functional pages, interactive components, and real-world website sections commonly used by organizations and non-profit projects.

## Pages Included
- Home (Greek & English version)
- About Us
- Events
- Donations
- Contact

## Key Features

### Homepage
- Available in Greek and English (English version uploaded)
- Image carousel showcasing featured content
- Structured layout with navigation and footer

### About Us
- Article-based content sections
- Clean content hierarchy and readable layout

### Events
- Dynamic events handling using JavaScript
- Past events with **"Read More"** functionality
- Upcoming events with **"Read More"** and **"Register"** buttons
- Separation between past and upcoming events

### Donations
- Donation flow with:
  - Fixed donation amounts
  - One-time and monthly donation options
  - Debit card payment selection
  - Continue button to proceed with the donation process

### Contact
- Full contact form
- Embedded map
- Footer with social media links

### Footer & Social Media
- Facebook
- Instagram
- Twitter
- Forum link

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes
This project was built entirely from scratch without CMS platforms, frameworks, or page builders.  
It is intended as a portfolio project to demonstrate hands-on front-end development, JavaScript logic, and real-world website structure.
